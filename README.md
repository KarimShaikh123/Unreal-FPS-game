# Unreal FPS Game
A first-person-shooter minigame built in Unreal Engine 5 using blueprint for scripting.

# About
Developed a fully functional FPS game from scratch using UE5's and other learning resources.
Serves as an introduction to game development, UE5's engine, and blueprint scripting.

# Features
- First-person-shooter mechanics
- Target practice mode with scoring system
- HUD and UI system including end screen
- Character movement with jumping, dashing, and wall jumping

# What I learned
- Blueprint visual scripting in Unreal Engine 5
- Game design and the components in it
- UI/UX design within a game engine
- Asset management

# Tools used
- Unreal Engine 5
- Blueprint scripting
- Unreal Engines's built in physics and animation systems
